# Conversation Log

## 設計決定事項

### 動画生成タイミング
- **決定**: 初回アクセス時に自動生成
- **理由**: ユーザーの手間を省き、即座にシミュレーション体験を提供するため

### Boidsパラメータ
- **決定**: 固定値を使用
- **理由**: 見栄えの良いプリセット値で固定し、UIを簡潔に保つため
- **設定値**:
  - 分離半径: 8px、重み: 1.5
  - 整列半径: 25px、重み: 1.0
  - 結合半径: 25px、重み: 1.0

### 技術スタック
- **Rust + Rayon**: 10万点の並列計算に最適
- **FFmpeg**: 高品質な動画エンコード
- **Actix-web + SSE**: リアルタイム進捗通知
- **空間分割グリッド**: O(n²) を O(n) に最適化
